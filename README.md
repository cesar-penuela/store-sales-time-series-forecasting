# Store Sales - Time Series Forecasting

## 📌 Project Overview
This project tackles the Kaggle competition:
**Store Sales - Time Series Forecasting**

The goal is to predict daily sales for a large Ecuadorian-based grocery retailer using historical data.

## 🎯 Objectives
- Perform exploratory data analysis (EDA)
- Build forecasting models
- Compare different machine learning approaches
- Optimize performance based on evaluation metrics

## 📂 Project Structure

```
store-sales/
│
├── data/               # Local data (ignored in Git)
├── notebooks/          # Exploratory analysis
├── src/                # Source code (pipelines, training scripts)
├── models/             # Trained models (ignored in Git)
├── README.md
└── .gitignore
```

## 📊 Dataset

This project uses data from the Kaggle competition:

Store Sales - Time Series Forecasting

To download the dataset:

```
kaggle competitions download -c store-sales-time-series-forecasting
```
